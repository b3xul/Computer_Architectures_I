Copiare i file contenuti nell'archivio come spiegato di seguito:
- emu8086.io e emu8086.hw vanno in C:\
- tutti gli altri file (inclusa la cartella lib) vanno nella sottocartella DEVICES contenuta nella cartella di installazione di emu8086.

Il sistema supportato basato su 8086 è illustrato nel file emu_system.ppt.

IMPORTANTE:
- I dispositivi 8255 e 8253 possono funzionare con o senza 8259, e la scelta deve essere effettuata agendo sul menù file dell'emulatore della periferica
- Per permettere il corretto funzionamento degli emulatori è necessario lanciare emu8086 con diritti di amministratore del sistema (clic destro sull'icona di emu8086, quindi "Esegui come amministratore").


=======================================================================================

Please copy the file in archive to your system as following:
- emu8086.io and emu8086.hw in C:\
- all other files (including the lib folder) in the subfolder DEVICES to be found within the emu8086 installation folder.

The supported system based on 8086 is illustrated in the emu_system.ppt chart.

IMPORTANT: 
- 8255 and 8253 can work with or without the 8259, which can be bypassed from the file menu
- In order for the emulators to work, in Windows Vista or 7 emu8086 needs to be started with administrator rights (right-click on emu8086 icon, "Run as administrator").
